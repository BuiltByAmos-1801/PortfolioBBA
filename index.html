Technical Skills

I combine strong technical knowledge with creative thinking to build interactive and scalable digital experiences. Each skill below represents hours of learning, experimentation, and hands-on work — all contributing to my growth as a modern web developer.

HTML5 & CSS3

The foundation of the web. I write clean, semantic HTML and modern CSS that brings designs to life. From flexbox to grid layouts, I ensure pixel-perfect implementation and responsiveness across all screen sizes. My CSS philosophy: minimal yet meaningful.

JavaScript (ES6+)

JavaScript adds life to the web, and it’s the language I breathe. I use ES6+ features to write efficient, modular, and maintainable code. From dynamic UI components to async API handling, I make sure every feature performs smoothly.

React.js

My go-to framework for building powerful, interactive frontends. I love the component-based architecture, hooks, and reusability it offers. I also focus on optimizing rendering, using state management libraries, and ensuring accessibility.

Node.js & Express

For backend development, I rely on Node.js and Express to build fast, lightweight, and scalable APIs. Whether it’s handling routes, authentication, or integrating databases, I ensure the backend complements the frontend perfectly.

MongoDB & SQL

Data is the backbone of any app. I use MongoDB for flexibility and MySQL for structure — depending on the project’s needs. I design efficient schemas, handle CRUD operations, and ensure data security and speed.

UI/UX Design

Designing is storytelling. I use Figma and Adobe XD to create intuitive user flows, wireframes, and prototypes that translate business goals into delightful experiences. My design mantra: clarity, consistency, and empathy.

I continuously upgrade my skill set by exploring frameworks like Next.js, Tailwind CSS, and TypeScript. For me, learning never stops — every project teaches something new, and that’s what makes this journey exciting.
